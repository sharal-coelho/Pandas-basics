# Working with Pandas in Python

This repository demonstrates **basic and commonly used Pandas operations** using a built-in dataset (Iris).

## Dataset
- Iris dataset from `sklearn.datasets`
- Converted into a Pandas DataFrame

## Pandas Operations Covered
- Creating DataFrame
- head(), tail()
- shape, columns, dtypes
- describe()
- isnull()
- Column selection
- Row filtering
- Adding new columns
- groupby()
- sort_values()
- value_counts()
- Exporting to CSV

## Technologies Used
- Python
- Pandas
- Scikit-learn

## Use Case
Ideal for beginners learning Pandas and for showcasing **data handling skills** on GitHub.
